 STEP 1: Create GitHub Repository
Go to GitHub
Create repo:
Name: simple-maven-app
Add README 
Folder structure (VERY IMPORTANT)
simple-maven-app/
│── pom.xml
└── src/
    ├── main/java/com/example/App.java
    └── test/java/com/example/AppTest.java
 STEP 2: Create pom.xml
<project xmlns="http://maven.apache.org/POM/4.0.0"
xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xsi:schemaLocation="http://maven.apache.org/POM/4.0.0
http://maven.apache.org/xsd/maven-4.0.0.xsd">

    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>simple-maven-app</artifactId>
    <version>1.0-SNAPSHOT</version>

    <dependencies>
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <version>4.13.2</version>
            <scope>test</scope>
        </dependency>
    </dependencies>

</project>
STEP 3: Create Java file (App.java)
Path:
src/main/java/com/example/App.java
package com.example;

public class App {
    public int add(int a, int b) {
        return a + b;
    }
}
STEP 4: Create Test file (AppTest.java)
Path:
src/test/java/com/example/AppTest.java
package com.example;

import org.junit.Test;
import static org.junit.Assert.*;

public class AppTest {

    @Test
    public void testAdd() {
        App app = new App();
        assertEquals(5, app.add(2, 3));
    }
}
STEP 5: Install Maven (LOCAL SYSTEM)
Install Apache Maven
Steps:
Download Maven ZIP
Extract to:
C:\Program Files\Apache\apache-maven-3.9.12
Set Environment Variable:
MAVEN_HOME = above path
Add to PATH:
C:\Program Files\Apache\apache-maven-3.9.12\bin
Verify:
mvn -version
STEP 6: Clone GitHub repo in VS Code
Open VS Code → Clone Repo:
https://github.com/your-username/simple-maven-app.git
Choose folder location
IMPORTANT NOTE
VS Code is ONLY for editing
Not for final build system
STEP 7: Run Maven locally
Open terminal inside project:
mvn clean test
✔ If success → BUILD SUCCESS
STEP 8: Configure Jenkins Maven
Go to Jenkins
Steps:
Manage Jenkins
Tools
Maven Installations
Add Maven
Name: M3
MAVEN_HOME: path (without /bin)
STEP 9: Create Jenkins Pipeline
New Item → Pipeline
Paste script below
Jenkinsfile (Pipeline Script)
pipeline {
    agent any

    tools {
        maven 'M3'
    }

    stages {

        stage('Checkout Git') {
            steps {
                git branch: 'main',
                url: 'https://github.com/your-username/simple-maven-app.git'
            }
        }

        stage('Build and Test') {
            steps {
                bat 'mvn clean test'
            }
        }
    }
}
STEP 10: Build Pipeline
In Jenkins:
Click Build Now
Go to Console Output
✔ You should see:
BUILD SUCCESS
TEST PASSED
